# FirstLight AI Studio – Copilot Instructions

## Mission

Build a production-grade AI Software Company platform.

## Technology

- Python 3.13
- FastAPI
- Next.js
- Google GenAI SDK
- Pydantic v2

## Rules

- Never change folder architecture.
- Never create duplicate folders.
- Never delete working code.
- Use SOLID principles.
- Production-ready code only.
- Use type hints.
- Add logging.
- Add exception handling.
- Never hardcode API keys.
- Read configuration from .env.
- Keep providers modular.
- Keep AI employees independent.

## Workflow

1. Analyze first.
2. Explain planned changes.
3. Modify only requested files.
4. Keep code clean.
5. Never leave TODO comments.
